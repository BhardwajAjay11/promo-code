== Description ==

My Promotional Banner Plugin helps you create and manage various promotional banners including popups, ticker banners, and header banners. It also integrates WooCommerce coupon banners to boost sales and user engagement.

**Key Features:**
- Display popup banners on any page
- Add WooCommerce coupons directly through banners
- Create scrolling tickers for headlines or offers
- Show header banners for important messages
- Easy-to-use admin panel for full control
- Fully responsive design for all devices

== Installation ==

1. Upload the `bannerPromo` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the "Promotional Banners" section in the WordPress admin dashboard to configure your banners.

== Frequently Asked Questions ==

= Is this plugin compatible with WooCommerce? =  
Yes, the plugin integrates seamlessly with WooCommerce.

= Can I customize the design of the banners? =  
Yes, you can use the plugin settings for basic customization, or override styles via your theme or child theme for more control.

= Where do I manage the banner settings? =  
From the WordPress dashboard, go to "Promotional Banners".
